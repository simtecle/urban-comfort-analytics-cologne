# Power BI Assets

Expected dashboard file:

```text
powerbi/urban_comfort_dashboard.pbix
```

Use Power BI Desktop in Import mode and connect to the prepared Supabase/PostgreSQL views:

- `mart.v_powerbi_city_part_overview`
- `mart.v_powerbi_score_breakdown`
- `mart.v_powerbi_data_quality`

The detailed build guide is in:

```text
docs/powerbi_build_guide.md
```

Screenshots for GitHub and portfolio use should be exported to:

```text
powerbi/screenshots/
```

Do not commit screenshots that show database credentials, connection strings, or personal account details.
